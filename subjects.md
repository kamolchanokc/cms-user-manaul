## 9.2 จัดการรายวิชากลาง

รายวิชากลาง คือคลังรายวิชามาตรฐานที่หลายหลักสูตรนำไปใช้ร่วมกันได้ ช่วยให้รหัส ชื่อ หน่วยกิต และคำอธิบายของวิชาเดียวกันตรงกันทุกที่ เมื่อคลิกเมนู **จัดการรายวิชากลาง** จะพบหน้าจัดการที่แบ่งรายวิชาออกเป็น 2 ประเภทหลัก (แท็บด้านบน)

### ประเภทของรายวิชากลาง

| ประเภท | คืออะไร |
| --- | --- |
| **รายวิชา GenEd (มหาวิทยาลัย)** | รายวิชาศึกษาทั่วไป (General Education) ระดับมหาวิทยาลัย ที่ทุกหลักสูตรใช้ร่วมกัน |
| **รายวิชาบริการ / รายวิชากลางคณะ** | รายวิชาที่คณะเปิดเป็นรายวิชากลาง/บริการ ให้หลักสูตรหรือคณะอื่นนำไปใช้ร่วม |

### การค้นหาและตัวกรอง

| ตัวกรอง | รายละเอียด |
| --- | --- |
| ค้นหา | พิมพ์ **รหัสวิชา** หรือ **ชื่อวิชา** |
| ประเภทวิชา / กลุ่ม GenEd | กรองตามประเภทวิชา หรือกลุ่ม GenEd |
| วิทยาเขต | เลือกวิทยาเขต (หรือ "ทุกวิทยาเขต") |
| สังกัด / คณะ | กรองตามคณะที่เป็นเจ้าของรายวิชา |

### การดำเนินการกับรายวิชากลาง

| ปุ่ม / เมนู | ใช้ทำอะไร |
| --- | --- |
| **เพิ่มรายวิชา** | สร้างรายวิชากลางใหม่ (แยกฟอร์มระหว่าง GenEd กับ รายวิชากลางคณะ) — กรอกรหัสวิชา · ชื่อ (ไทย/อังกฤษ) · หน่วยกิต · รูปแบบหน่วยกิต · คำอธิบาย (ไทย/อังกฤษ) · ประเภทวิชา · ชั่วโมงบรรยาย/ปฏิบัติ/ศึกษาด้วยตนเอง · กลุ่ม GenEd/สังกัด |
| **แก้ไข** | แก้ไขข้อมูลรายวิชาที่มีอยู่ |
| **ปรับปรุงรายวิชา** | สร้าง "คำขอปรับปรุง" รายวิชา (ผ่านกระบวนการตรวจสอบ/อนุมัติ) — ปรับปรุงหลายรายการพร้อมกันได้ (Bulk Revise) |
| **นำเข้าข้อมูล** | นำเข้ารายวิชาจากไฟล์ **Excel** (มี Template ให้ดาวน์โหลด) |
| **ลบรายวิชา** | ลบรายวิชา (มีกล่องยืนยัน) |
| **เปรียบเทียบ** | ดูความเปลี่ยนแปลงก่อน–หลังปรับปรุง (Compare) |

### กระบวนการตรวจสอบ/อนุมัติรายวิชากลาง

รายวิชากลาง (ทั้ง GenEd และกลางคณะ) มีกระบวนการทำงานเป็นสถานะ ไม่ใช่แก้แล้วมีผลทันทีทุกกรณี

```mermaid
flowchart LR
    A["รอดำเนินการ<br/>(คำขอเพิ่ม/ปรับปรุง/ปิด-เปิดใช้งาน)"] --> B["ตรวจสอบ / อนุมัติ"]
    B -->|อนุมัติ| C["เปิดใช้งาน"]
    B -->|ปฏิเสธ (ระบุเหตุผล)| A
    C -->|ขอปรับปรุง| A
    C -->|ขอปิดใช้งาน| D["ปิดใช้งาน"]
```

* **รอดำเนินการ** — คำขอเพิ่ม/ปรับปรุง/ปิด-เปิดใช้งาน ที่ยังไม่ได้ตรวจ
* **ตรวจสอบ / อนุมัติ** — ผู้มีสิทธิ์ตรวจแล้ว **อนุมัติ** หรือ **ปฏิเสธ (พร้อมระบุเหตุผล)**
* **เปิดใช้งาน / ปิดใช้งาน** — รายวิชาที่ใช้งานได้ / ระงับการใช้งาน
* สามารถ **ยกเลิกคำขอ** ที่ส่งไปแล้วได้ (มีกล่องยืนยัน)

> ⚠️ **ผลกระทบเป็นวงกว้าง:** เนื่องจากรายวิชากลางถูกอ้างอิงจากหลายหลักสูตร การแก้ไขรหัส ชื่อ หรือหน่วยกิตจะกระทบทุกหลักสูตรที่ใช้รายวิชานั้น ก่อนแก้ควรตรวจสอบว่ามีหลักสูตรใดใช้อยู่บ้าง (ดูได้จากปุ่ม **สังกัด/หลักสูตรที่ใช้งาน**) และยืนยันว่าการเปลี่ยนแปลงนี้สมควรมีผลกับทุกหลักสูตร หากต้องการเปลี่ยนเฉพาะหลักสูตรเดียว ควรพิจารณาสร้างรายวิชาแยกแทน

> ℹ️ รายวิชาที่ยังมีหลักสูตรใช้งานอยู่ อาจ **ลบไม่ได้** (ระบบจะแจ้ง "ไม่สามารถลบรายวิชาได้") ต้องนำออกจากหลักสูตรที่ใช้ก่อน
