# 8. การปิดหลักสูตร

ใช้เมื่อต้องการปิดหลักสูตรอย่างเป็นทางการ หลักสูตรที่ปิดแล้วจะไม่รับนักศึกษาใหม่ แต่นักศึกษาเดิมยังคงสามารถเรียนต่อจนจบได้

กระบวนการแบ่งเป็น 2 ขั้นตอนหลัก

{% code overflow="wrap" %}
```
[ขั้น 1: ปิดหลักสูตร] → [ขั้น 2: การอนุมัติหลักสูตร]
```
{% endcode %}

## ความต่างระหว่าง "ระงับการรับนักศึกษา" กับ "ปิดดำเนินการ"

การปิดหลักสูตรมักมีสองช่วง

1. ระงับการรับนักศึกษา — หยุดรับนักศึกษาใหม่ แต่ยังมีนักศึกษาเก่าเรียนอยู่ (สถานะ "ระงับการรับนักศึกษา")
2. ปิดดำเนินการ — เมื่อนักศึกษาทุกคนสำเร็จการศึกษาหรือพ้นสภาพแล้ว จึงปิดหลักสูตรอย่างสมบูรณ์ (สถานะ "ปิดดำเนินการ")

## ขั้นตอนการดำเนินงาน

1. ไปที่เมนู จัดการหลักสูตร
2. ค้นหาและเลือกหลักสูตรที่ต้องการปิด
3. คลิกดำเนินการ ปิดหลักสูตร
4. ข้อมูลนักศึกษาคงค้าง ที่ยังไม่สำเร็จการศึกษา — ระบบอาจตรวจสอบและแสดงจำนวนคงค้างให้ยืนยัน
5. กรอกข้อมูลที่เกี่ยวข้อง เช่น เหตุผลการปิดหลักสูตร — ระบุเหตุผลให้ชัดเจน (เช่น ปรับปรุงเป็นหลักสูตรใหม่ ความต้องการลดลง)

1. แผนดูแลนักศึกษา หลังปิดหลักสูตร — แผนรองรับให้นักศึกษาเดิมเรียนจบได้
2. ส่งกระบวนการอนุมัติปิดหลักสูตร ซึ่งประกอบด้วย 4 แท็บ (ใช้ลำดับการอนุมัติชุดเดียวกับการเพิ่ม/ปรับปรุงหลักสูตร แต่ใช้เฉพาะลำดับ 1–2, 4, 5, 6)

| แท็บ   | ลำดับการอนุมัติ                   | ผู้พิจารณา / หน่วยงาน                                        |
| ------ | --------------------------------- | ------------------------------------------------------------ |
| แท็บ 1 | ลำดับ 1 และ 2 (รวมไว้ในแท็บเดียว) | คณะอนุกรรมการกลั่นกรอง PLOs และ คณะกรรมการประจำคณะ / ส่วนงาน |
| แท็บ 2 | ลำดับ 4                           | คณะกรรมการนโยบายการพัฒนาทรัพยากรมนุษย์และการจัดการศึกษา      |
| แท็บ 3 | ลำดับ 5                           | สภามหาวิทยาลัย                                               |
| แท็บ 4 | ลำดับ 6                           | สป.อว.                                                       |

> ⚠️ **การปิดดำเนินการอัตโนมัติ:** ระบบมีกลไกตรวจสอบและช่วยเปลี่ยนสถานะหลักสูตรที่เข้าเงื่อนไขการปิด/ระงับ พร้อมการแจ้งเตือนที่เกี่ยวข้อง ผู้รับผิดชอบควรติดตามการแจ้งเตือนเหล่านี้เพื่อยืนยันความถูกต้อง
