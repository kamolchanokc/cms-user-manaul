# 13. คำอธิบายคำศัพท์

| คำ/ตัวย่อ          | ความหมาย                                                                                                                                 |
| ------------------ | ---------------------------------------------------------------------------------------------------------------------------------------- |
| CMS                | Curriculum Management System — ระบบบริหารจัดการหลักสูตร                                                                                  |
| รหัส CISA          | รหัสประจำหลักสูตร 14 หลัก ที่ใช้อ้างอิงทั่วทั้งระบบ                                                                                      |
| PLO                | Program Learning Outcomes — ผลลัพธ์การเรียนรู้ระดับ **หลักสูตร**                                                                         |
| MLO                | ผลลัพธ์การเรียนรู้ระดับ **กลุ่มวิชาเอก/วิชาโท/กลุ่มสาขาวิชา** (PLOs ที่ผูกกับกลุ่มวิชาเอก)                                               |
| sub-PLO            | ข้อย่อยภายใต้ PLO แต่ละข้อ                                                                                                               |
| CLO                | Course Learning Outcomes — ผลลัพธ์การเรียนรู้ระดับ **รายวิชา**                                                                           |
| Mapping            | การจับคู่ความสัมพันธ์ระหว่าง PLO กับรายวิชา เพื่อแสดงว่าวิชาใดส่งผลต่อ PLO ใด                                                            |
| ชุด PLOs           | กลุ่ม PLO หนึ่งชุด — หลักสูตรมีได้แบบชุดเดียว หรือมากกว่าหนึ่งชุด (แยกตามกลุ่มวิชาเอก/สาขา)                                              |
| ปีหน้าปก           | ปีอ้างอิงของ "รุ่น" หลักสูตรที่ระบุบนหน้าปกเล่ม                                                                                          |
| หลักสูตรแม่–ลูก    | หลักสูตรเดิม (แม่) และหลักสูตรปรับปรุงที่อ้างอิงกลับไป (ลูก)                                                                             |
| ลำดับการอนุมัติ    | ขั้นพิจารณาอนุมัติหลักสูตร 8 ลำดับ (ดูหัวข้อ [5. การเพิ่มหลักสูตรใหม่](add-curriculum.md)) — ลำดับ 5 = สภามหาวิทยาลัย คือจุดเปลี่ยนสถานะ |
| คราวประชุมครั้งที่ | ครั้ง/วันที่ของการประชุมที่พิจารณาเห็นชอบในแต่ละลำดับการอนุมัติ                                                                          |
| OKR                | Objectives and Key Results — ตัวชี้วัดความสำเร็จของหลักสูตร                                                                              |
| CWIE               | Cooperative and Work Integrated Education — สหกิจศึกษาและการศึกษาเชิงบูรณาการกับการทำงาน                                                 |
| ISCED              | มาตรฐานสากลการจัดประเภทการศึกษา (International Standard Classification of Education)                                                     |
| สป.อว.             | สำนักงานปลัดกระทรวงการอุดมศึกษา วิทยาศาสตร์ วิจัยและนวัตกรรม                                                                             |
| ก.พ.               | สำนักงานคณะกรรมการข้าราชการพลเรือน — รับรองคุณวุฒิ                                                                                       |
| ก.ค.ศ.             | สำนักงานคณะกรรมการข้าราชการครูและบุคลากรทางการศึกษา — รับรองคุณวุฒิ                                                                      |
| Sandbox            | หลักสูตรที่จัดการศึกษาแตกต่างจากมาตรฐาน (Higher Education Sandbox)                                                                       |
| Skill Transcript   | เอกสารแสดงทักษะ/สมรรถนะที่นักศึกษาได้รับ                                                                                                 |
| Stepper            | แถบแสดงลำดับขั้นตอนด้านบนของหน้าจอ                                                                                                       |
