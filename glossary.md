# อภิธานศัพท์ (Glossary)

| คำ/ตัวย่อ | ความหมาย |
| --- | --- |
| CMS | Curriculum Management System — ระบบบริหารจัดการหลักสูตร |
| รหัส CISA | รหัสประจำหลักสูตรที่ใช้อ้างอิงทั่วทั้งระบบ |
| PLO | Program Learning Outcomes — ผลลัพธ์การเรียนรู้ระดับหลักสูตร |
| CLO | Course Learning Outcomes — ผลลัพธ์การเรียนรู้ระดับรายวิชา |
| Mapping | การจับคู่ความสัมพันธ์ระหว่าง PLO กับรายวิชา เพื่อแสดงว่าวิชาใดส่งผลต่อ PLO ใด |
| ปีหน้าปก | ปีอ้างอิงของ "รุ่น" หลักสูตรที่ระบุบนหน้าปกเล่ม |
| หลักสูตรแม่–ลูก | หลักสูตรเดิม (แม่) และหลักสูตรปรับปรุงที่อ้างอิงกลับไป (ลูก) |
| Stepper | แถบแสดงลำดับขั้นตอนด้านบนของหน้าจอ |
