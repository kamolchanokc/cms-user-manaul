# 10. การตรวจสอบข้อมูลอาจารย์

1. คลิกเมนู ตรวจสอบข้อมูลอาจารย์
2. ระบบซิงค์และนำข้อมูลอาจารย์มาแสดงจากระบบ **HRMIS**

**ข้อมูลที่ดูได้**

| หัวข้อ            | รายละเอียด                                                  |
| ----------------- | ----------------------------------------------------------- |
| ข้อมูลทั่วไป      | ชื่อ-สกุล ตำแหน่ง คณะ/ภาควิชา วิทยาเขต                      |
| คุณวุฒิการศึกษา   | ระดับการศึกษาสูงสุด สาขาวิชา สถาบัน ปีที่จบ                 |
| ตำแหน่งทางวิชาการ | อาจารย์ / ผู้ช่วยศาสตราจารย์ / รองศาสตราจารย์ / ศาสตราจารย์ |
| ภาระการสอน        | รายวิชาที่รับผิดชอบในแต่ละภาคการศึกษา                       |
| ผลงานทางวิชาการ   | งานวิจัย บทความ ตำรา (ซิงค์มาจากระบบ HRMIS)                 |

{% endhint %}

## การตรวจคุณสมบัติ 🟢อาจารย์ประจำหลักสูตร

ระบบใช้**ข้อมูลคุณวุฒิและตำแหน่งทางวิชาการ**นี้ในการ ตรวจคุณสมบัติอาจารย์ประจำหลักสูตร ตามเกณฑ์มาตรฐาน เมื่อมีอาจารย์ที่คุณวุฒิไม่ผ่านเกณฑ์ในหลักสูตรใด ระบบจะแสดงในแท็บอาจารย์ของ Dashboard พร้อมข้อมูลสังกัด (วิทยาเขต/คณะ/ภาควิชา) เพื่อให้ติดตามแก้ไขได้ตรงคน

ข้อมูลอาจารย์ (ชื่อ-สกุล คุณวุฒิ ตำแหน่งทางวิชาการ และผลงานทางวิชาการ) มาจากระบบ **HRMIS** หากพบว่าข้อมูลไม่ถูกต้อง ส่วนใหญ่ต้องแก้ที่ **HRMIS ต้นทาง** แล้วข้อมูลจะซิงค์กลับมาที่ CMS ไม่ใช่แก้ที่ CMS โดยตรง
