**ค้นหาและถาม AI ได้**

* 🔍 **พิมพ์ค้นหา** — กดช่องค้นหา (Search) ที่มุมบนของคู่มือ หรือกด `Ctrl + K` เพื่อค้นหาหัวข้อ/คำที่ต้องการได้ทั้งเล่ม
